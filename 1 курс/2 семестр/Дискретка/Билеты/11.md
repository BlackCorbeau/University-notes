# –î–≤–æ–π—Å—Ç–≤–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è
**–î–≤–æ–π—Å—Ç–≤–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è** $f^*$ - $f^*(x_1, x_2, \dots, x_n) = \overline{f(\overline{x_1}, \overline{x_2}, \dots, \overline{x_n})}$
$(f^*)^* = f$

–ü—Ä–∏–º–µ—Ä—ã:
- $0^* = 1$
- $x^* = x$
- $\bar x^* = \bar x$
- $(xy)^* = x \vee y$
- $(x \oplus y)^* = x \equiv y$
- $(x|y)^* = x \downarrow y$
- $(x \rightarrow y)^* = y > x$

# –ü—Ä–∏–Ω—Ü–∏–ø –¥–≤–æ–π—Å—Ç–≤–µ–Ω–Ω–æ—Å—Ç–∏
## –¢–µ–æ—Ä–µ–º–∞
–ü—É—Å—Ç—å $f(x_1, x_2, \dots, x_n) –∏ g(y_1, y_2, \dots, y_m)$ - –ª–æ–≥–∏—á–µ—Å–∫–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏ –∏ $h = f(x_1, x_2, \dots, x_{k-1}, g(x_1, y_2, \dots, y_m), x_{k+1}, \dots, x_n)$
–¢–æ–≥–¥–∞ $h^* = f^*(x_1, x_2, \dots, x_{k-1}, g^*(y_1, y_2, \dots, y_m), x_{k+1}, \dots, x_n)$
## –î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ
–ù–£–û $k = n$: $h(x_1, x_2, \dots, x_{n-1}, y_1, y_2, \dots, y_m) = f(x_1, x_2, \dots, x_{n-1}, g(y_1, y_2, \dots, y_m))$

–ü–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—é –¥–≤–æ–π—Å—Ç–≤–µ–Ω–Ω–æ—Å—Ç–∏, $h^*(x_1, x_2, \dots, x_{n-1}, y_1, y_2, \dots, y_m) = \overline{h(\overline{x_1}, \overline{x_2}, \dots, \overline{x_{n-1}}, \overline{y_1}, \overline{y_2}, \dots, \overline{y_m})} = \overline{f(\overline{x_1}, \overline{x_2}, \dots, \overline{x_{n-1}}, g(\overline{y_1}, \overline{y_2}, \dots, \overline{y_m}))}$
$g(\overline{y_1}, \overline{y_2}, \dots, \overline{y_m}) = \overline{g^*(y_1, y_2, \dots, y_m)}$, –ø–æ—ç—Ç–æ–º—É $h^* = \overline{f\left(\overline{x_1}, \overline{x_2}, \dots, \overline{x_{n-1}}, \overline{g^*(y_1, y_2, \dots, y_m)}\right)} = f^*(x_1, x_2, \dots, x_{n-1}, g^*(y_1, y_2, \dots, y_m))$

## –°–ª–µ–¥—Å—Ç–≤–∏–µ
> –ü—É—Å—Ç—å —Ñ—É–Ω–∫—Ü–∏—è ùëì –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∞ –Ω–µ–∫–æ—Ç–æ—Ä–æ–π —Ñ–æ—Ä–º—É–ª–æ–π/—Å—Ö–µ–º–æ–π. –ß—Ç–æ–±—ã –ø–æ–ª—É—á–∏—Ç—å —Ñ–æ—Ä–º—É–ª—É/—Å—Ö–µ–º—É, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—â—É—é —Ñ—É–Ω–∫—Ü–∏—é $ùëì^‚àó$, –Ω—É–∂–Ω–æ –∑–∞–º–µ–Ω–∏—Ç—å –≤ —Ñ–æ—Ä–º—É–ª–µ –≤—Å–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –∏ –∫–æ–Ω—Å—Ç–∞–Ω—Ç—ã / —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã –Ω–∞ –¥–≤–æ–π—Å—Ç–≤–µ–Ω–Ω—ã–µ –∏–º.

# –ó–∞–º–∫–Ω—É—Ç–æ—Å—Ç—å –∫–ª–∞—Å—Å–∞ ùëÜ
## –¢–µ–æ—Ä–µ–º–∞
–ö–ª–∞—Å—Å S –∑–∞–º–∫–Ω—É—Ç
## –î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ
–ü—É—Å—Ç—å $f(x_1, x_2, \dots, x_n) \in S$ –∏ $g(y_1, y_2, \dots, y_m) \in S$
–†–∞—Å—Å–º–æ—Ç—Ä–∏–º $h(x_1, x_2, \dots, x_{n-1}, y_1, y_2, \dots, y_m) = f(x_1, x_2, \dots, x_{n-1}, g(y_1, y_2, \dots, y_m))$
–ò–∑ –ø—Ä–∏–Ω—Ü–∏–ø–∞ –¥–≤–æ–π—Å—Ç–≤–µ–Ω–Ω–æ—Å—Ç–∏, $h^* = f^*(x_1, x_2, \dots, x_{n-1}, g^*(y_1, y_2, \dots, y_m)) = f(x_1, x_2, \dots, x_{n-1}, g(y_1, y_2, \dots, y_m)) = h$, —Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, $h \in S$

# –õ–µ–º–º–∞ –æ –Ω–µ—Å–∞–º–æ–¥–≤–æ–π—Å—Ç–≤–µ–Ω–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏
## –õ–µ–º–º–∞
–ï—Å–ª–∏ —Ñ—É–Ω–∫—Ü–∏—è f –Ω–µ—Å–∞–º–æ–¥–≤–æ–π—Å—Ç–≤–µ–Ω–Ω–∞, —Ç–æ –∫–æ–Ω—Å—Ç–∞–Ω—Ç—ã —è–≤–ª—è—é—Ç—Å—è —Å—É–ø–µ—Ä–ø–æ–∑–∏—Ü–∏–µ–π —Ñ—É–Ω–∫—Ü–∏–π $f$ –∏ $\bar x$. –¢.–µ. –µ—Å–ª–∏ $f \notin S$, —Ç–æ ${0,1} \subseteq [\{f, \bar x\}]$
## –î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ
–ü—É—Å—Ç—å $f(x_1, x_2, \dots, x_n) \notin S$. –¢–æ–≥–¥–∞ —Å—É—â–µ—Å—Ç–≤—É–µ—Ç —Ç–∞–∫–æ–π –Ω–∞–±–æ—Ä $(\alpha_1, \alpha_2, \dots, \alpha_n)$, —á—Ç–æ $f(\alpha_1, \alpha_2, \dots, \alpha_n) = f(\bar\alpha_1, \bar\alpha_2, \dots, \bar\alpha_n) = c \in \{0,1\}$ ([[1 –∫—É—Ä—Å/2 —Å–µ–º–µ—Å—Ç—Ä/–î–∏—Å–∫—Ä–µ—Ç–∫–∞/–ë–∏–ª–µ—Ç—ã/3#–†–∞–∑–ª–æ–∂–µ–Ω–∏–µ —Ñ—É–Ω–∫—Ü–∏–∏ –ø–æ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π|–†–∞–∑–ª–æ–∂–µ–Ω–∏–µ –ø–æ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π]])

$h(x) = f(x^{\alpha_1}, x^{\alpha_2}, \dots, x^{\alpha_n})$
